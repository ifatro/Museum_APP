# Museum Card Scanner (Local, Web)

אפליקציית ווב שמריצה OCR **מקומית בדפדפן** (ללא העלאת תמונות לשרת).

## הפעלה

פתח/י את `index.html` בדפדפן (עדיף Chrome/Edge).

## מה יש כרגע (MVP)

- העלאת תמונה / drag & drop
- OCR על כל התמונה (עברית+אנגלית) באמצעות `tesseract.js`
- ייצוא JSON ו-CSV

## הצעד הבא (אחרי שתשלח/י דוגמת כרטיסייה)

כי כל הכרטיסיות באותה תבנית, נחליף את OCR “על כל התמונה” ל־OCR לפי אזורים קבועים (ROI) עבור שדות כמו:

- שם פריט
- אמן/יוצר
- תאריך/תקופה
- חומרים/טכניקה
- מידות
- מספר קטלוג
- תיאור

זה משפר מאוד דיוק ומהירות.

## פרטיות

אין העלאה של תמונות לשרת.  
שים/י לב: כרגע הספרייה `tesseract.js` נטענת מ־CDN והמודל/שפות עשויים להיטען אוטומטית בזמן ריצה. אם נדרש offline מוחלט, נארוז הכל מקומית בפרויקט (בלי תלות חיצונית).
